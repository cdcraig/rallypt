# RallyPT

Secure communications platform for law enforcement. Signal-like UX with compliance, records management, and real-time coordination features built for departments.

## Tech Stack

- **Frontend:** Vite + React + TypeScript
- **Mobile:** Capacitor (iOS + Android)
- **Backend:** Supabase

## Key Features

- Encrypted messaging with department-level compliance records
- Going Live — real-time officer location on map
- Canvas — persistent location trails for pursuits and incident tracking

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Mobile

```bash
npx cap sync
npx cap open ios
npx cap open android
```
